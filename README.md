# EVM in Java

This is a standalone EVM implementation derived from [EthereumJ v1.8.1](https://github.com/ethereum/ethereumj/tree/1.8.1), with
the following targets:
- Light weight
- Embeddable
- High performance


## Build

```
mvn install
```


## Format

```
mvn formatter:format license:format findbugs:check
```
